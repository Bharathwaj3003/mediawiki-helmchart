OS platform used Ubuntu 22.04


step 1: Install docker-desktop
step 2: Enable kubernetes in docker-desktop
step 3: Install kubectl and kubectx
step 4: Install helm chart
step 5: Install ingress-nginx from official kubernetes documentation
https://github.com/kubernetes/ingress-nginx/blob/main/docs/deploy/index.md
step 6: Build Docker image for the mediawiki application by running the following commands
cd app;
docker build . -t mediawiki:1.0
step 7: Build Docker image for the mariadb database by running the following commands
cd db;
docker build . -t mediawiki-db:1.0
